如果优雅的"停止"一个正在运行的JAVA线程

发表于 2016-04-27   |  

最近在开发过程中，遇到一个需求

  • 当用户输完快递单号后，发起请求，自动补全手机号码
  • 请求超过5秒后结束请求，并提示用户没找到手机号

第一发

很简单的需求，是吧。然后我就啪啪啪写下了代码。

public void request() {

    thread = new Thread(new Runnable() {
        @Override
        public void run() {
            // 请求网络
            // ...
            // ...
            // 返回结果请求To UI
        }
    });

    thread.start();

    new Handler().postDelayed(new Runnable() {
        @Override
        public void run() {
            thread.stop();
        }
    }, 5 * 1000);

}

看起来是这么的完美。不过这个stop上面有一根删除线是什么鬼。它的文档是这样描述的。
